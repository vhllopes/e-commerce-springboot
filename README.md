# API FlaBijoux - Spring Boot

API REST para o sistema de gerenciamento da loja de bijuterias FlaBijoux.

## Tecnologias Utilizadas
- Java 17+
- Spring Boot 3
- Maven
- MySQL

## Configuração do Banco de Dados
1. Certifique-se de ter o MySQL instalado e rodando.
2. Crie um banco de dados chamado `db_flabijoux`.
3. Execute o script localizado em `db/schema.sql` para criar todas as tabelas necessárias.
